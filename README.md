# Multiple Disease Prediction System

A Machine Learning system that predicts 3 diseases:
- Diabetes (Random Forest - 72% accuracy)
- Heart Disease (Random Forest - 76% accuracy)
- Neurological Disorder (SVM - 53% accuracy)

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn

## How to Run
1. Clone the repo
2. Install requirements: pip install -r requirements.txt
3. Open the notebook
